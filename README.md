A lightweight **retro arcade Snake game** built using pure **HTML, CSS and JavaScript**.

Play directly in the browser with smooth gameplay, retro sounds and modern features.

---

## 🎮 Features

✨ Classic Snake gameplay
🏆 Achievement system
🎵 Retro procedural music
📱 Mobile support (touch controls)
⚡ High performance Canvas rendering
💾 High score saved with LocalStorage
💰 Space for Google AdSense integration

---

## 🕹️ Controls

**Desktop**

⬅️ ➡️ ⬆️ ⬇️ Arrow Keys to move

**Mobile**

👉 Swipe gestures or touch controls

---

## 🚀 Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Canvas API
* Web Audio API

---

## 📦 Run the Game

1. Download the project
2. Open **index.html** in your browser
3. Play instantly 🎮

No installation required.
